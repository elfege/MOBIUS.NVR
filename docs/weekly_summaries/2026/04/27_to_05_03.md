## Decisions made

- **Speech-content categorization (insults / arguments / harassment / nagging) is OUT OF SCOPE at intake.** Transcribed and ingested into Anamnesis; user retrieves and judges. False-positive auto-classification is evidentially harmful.
- **Voice cloning / deepfake-of-text is OUT OF SCOPE.** This corpus exists for evidence, not training.
- **mp4 video is NOT captured separately by the pipeline.** Existing recording stack already produces mp4s; manifest entries reference time-ranges, case-binding daemon resolves to mp4s at promotion time.
- **Universal-user product framing:** case-binding via HTTP API; `server:~/0_LEGAL` is one consumer among potentially many.
- **PostgREST anon access for evidence tables:** matches existing project convention. To be tightened in Phase 5 when the formal `/api/evidence/` Flask surface introduces proper auth.
