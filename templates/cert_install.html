iPhone / iPad Detected
Install Profile
1 Tap Install Profile above
2 When prompted, tap Allow
3 Go to Settings → General → VPN & Device Management
4 Tap the Home NVR profile → Install
5 Go to Settings → General → About → Certificate Trust Settings
6 Enable full trust for NVR Local CA
macOS Detected
Download Certificate
1 Click Download Certificate above
2 Double-click the downloaded .crt file
3 Keychain Access opens — add to System keychain
4 Double-click the cert in Keychain → Trust → set to Always Trust
Windows Detected
Download Certificate
1 Click Download Certificate above
2 Double-click the downloaded .crt file
3 Click Install Certificate
4 Select Local Machine → Next
5 Choose Place all certificates in the following store
6 Browse → Trusted Root Certification Authorities → OK → Finish
Linux Detected
Download Certificate
1 Click Download Certificate above
2 Copy to trust store:
sudo cp NVR_Local_CA.crt /usr/local/share/ca-certificates/
3 Update certificates:
sudo update-ca-certificates
4 Chrome/Chromium also needs:
certutil -d sql:$HOME/.pki/nssdb -A -t "C,," -n "NVR Local CA" -i NVR_Local_CA.crt
5 Firefox: Settings → Privacy → Certificates → Import → select the .crt → check "Trust to identify websites"
Android Detected
Download Certificate
1 Tap Download Certificate above
2 Open the downloaded file (or go to Settings → Security → Install from storage)
3 Name it NVR Local CA
4 Select CA certificate when asked for credential use
5 Tap Install anyway on the warning prompt
